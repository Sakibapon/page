## Project Overview

This project develops a real-time action recognition system capable of detecting physical bullying from video footage using deep neural networks.

## Key Features

- Can detect physical bullying in real time
- Uses DNN with five different image classification models to detect actions from video footage
- Achieves high accuracy in action classification

## Technologies Used

- Deep Neural Networks
- Transfer Learning
- Computer Vision
- Python, TensorFlow/Keras
