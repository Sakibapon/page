## Project Overview

This project implements YOLO V4 for custom object detection with additional functionality including tracking, counting, OCR, and more.

## Key Features

- Detect, track, and count objects in real-time
- Print object information
- Apply Tesseract OCR for text recognition
- Crop detections and save as new images
- License plate recognition using Tesseract OCR
- Works on both videos and images

## Technologies Used

- YOLO V4
- Darknet
- DeepSort
- TensorFlow
- Flask
- Tesseract OCR
- Python
