## Project Overview

A web application for car parts and services where users can request services and track their order status through an intuitive interface.

## Key Features

### User Features
- Request services
- View previous request history
- Check status (pending, ongoing, or done)

### Admin Panel (Restricted Access)
- Add new services
- Delete existing services
- Update customer order status
- Manage service catalog

## Technologies Used

- HTML, CSS, JavaScript
- PHP
- MySQL
- Bootstrap
