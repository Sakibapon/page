## Project Overview

A comprehensive web application where users can explore universities, save favorites, register for consultations, and track their application status.

## Key Features

### User Features
- Save universities to favorites
- Register for consultations
- Check saved universities
- View previous consultations
- See assigned consultant
- Track consultation status

### Admin Panel (Restricted Access)
- Add new universities
- Remove universities
- Modify university information
- Change consultation status
- Assign consultants to students

## Technologies Used

- HTML, CSS, JavaScript
- PHP
- MySQL
- Bootstrap
